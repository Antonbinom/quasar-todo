## Описание

Тренировочный проект - Список дел

## Функционал

- Переход по страницам без перезагрузки
- Добавить задачу
- Удалить задачу
- Пометить как выполненную
- Сохранение задач после перезагрузки
- Адаптивный дизайн

## Стек технологий

- Vue3
- Vue-router
- Quasar Framework
- Компоненты Quasar
- localStorage

## Запуск quasar dev
## Сборка проекта quasar build
